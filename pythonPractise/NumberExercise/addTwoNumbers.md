# Exercise:

Given two numbers num1 and num2. The task is to write a Python program to find the addition of these two numbers.

### Examples:
```
Input: num1 = 5, num2 = 3
Output: 8

Input: num1 = 13, num2 = 6
Output: 19
```
# Helen Solution
```
a=int(input("Input a first number:"))
b=int(input("Input a second number:"))
print(a+b)
```
