# Exercise: 
Given a string of odd length greater 7, return a string made of the middle three chars of a given String

## Case 1:

- str1 = "JhonDipPeta"
- Expected Output:

- `Dip`
## Case 2:

- str2 = "JaSonAy"
- Expected Output:

- `Son`
# Helen Solution
```
a='JhonDipPeta'
print(a[4:7])
b="JaSonAy"
print(b[2:5])
```
